# Knowledge map of head complexity $H^{*}(f)$

*Synthesis of what is established about $H^{*}(f)$ (minimum attention heads to compute $f$ in the one-layer model). Every "proved" item is a verified lemma in `lemmas/` (informal-proof-checker `\boxed{1}` or Lean). Computational/conjectural items are flagged. AI-authored reference under `claude-comments/`; modifies no proof file.*

## 0. The object

$H^{*}(f)$ = least number of attention heads in the one-layer attention-only model (with linear readout from the query token) that computes $f : \lbrace 0,1\rbrace^n \to \lbrace 0,1\rbrace$ exactly.

## 1. Normal forms (what one/many heads compute)

- **One head = one linear-fractional atom** (L10): the scalar a head contributes is $\phi(x) = N(x)/D(x)$ with $D(x) = \gamma + \sum_i \rho_i\alpha^{x_i}$, $\gamma,\rho_i,\alpha > 0$.
- **Atom dictionary** (L13): expanding $\alpha^{x_i} = 1 + (\alpha-1)x_i$, $N, D$ are **affine**, $D > 0$ on the cube, and $D$'s slopes $\rho_i(\alpha-1)$ all share one sign — the **monotone bias** (a single shared $\alpha$, because $e_1 - e_0$ is position-independent). Conversely any affine $N$ over a one-sided positive affine $D$ is an atom.
- **Cleared (tangential-Chow) normal form** (L16): $H^{*}(f) = \mathrm{MFdeg}_{\pm}(f)$ = least $H$ such that $f$ is sign-represented by $P = \theta\prod_h D_h + \sum_h N_h\prod_{g\neq h}D_g$ with admissible affine pairs $(N_h, D_h)$. $P$ is a tangent vector at $\prod_h D_h$ to the variety of products of $H$ affine forms.

## 2. The placement of $H^{*}$ among classical invariants

$$
\boxed{\ \deg_{\pm}(f) \;\le\; \mathrm{tChow}_{\pm}(f) \;\le\; H^{*}(f) = \mathrm{MFdeg}_{\pm}(f) \;\le\; M_{+}(f) - 1\ }
$$

- $\deg_{\pm} \le H^{*}$ (L6); $\deg_{\pm} \le \mathrm{tChow}_{\pm} \le H^{*}$ (L18), where $\mathrm{tChow}_{\pm}$ drops positivity/one-sided constraints.
- $H^{*} \le M_{+} - 1$ (L9), $M_{+}$ = image size of a positive weighted sum.
- **$H^{*}$ is strictly finer than threshold degree** (L23): some $\deg_{\pm}=2$ function needs $H^{*} = \Omega(n)$ heads, by Warren counting (#$\lbrace H^{*}\le H\rbrace \le 2^{O(Hn^2)}$ vs $2^{\Theta(n^3)}$ degree-2 PTFs). The separation already holds without positivity (L24, on $\mathrm{tChow}_{\pm}$), so positivity is not its cause. So **$H^{*}$ is a genuinely new measure**, not a function of $\deg_{\pm}$.

## 4. Upper-bound toolkit

- Positive weighted-sum image: $H^{*} \le M_{+} - 1$ (L9).
- Calibrated interpolation: any real table over a positive weighted sum = const + $(M-1)$ atoms (L19); additive composition.
- Sign-change bound: $f = F(t)$, $t$ positive weighted sum $\Rightarrow H^{*} \le C(F)$ (L25).
- Simple invariant $A_{+}(f)$ = min alternations of $f$ over positive linear orderings; $H^{*} \le A_{+}$, exact for symmetric (L26).
- Monotone-term ($s$-term single-polarity) DNF: $H^{*} \le s$ (L14); sparse single-polarity threshold density: $H^{*} \le s$ (L20).

## 5. Lower-bound toolkit

- $\deg_{\pm}(f) \le H^{*}(f)$ (L6).
- Restriction monotonicity (L17): fixing coordinates can't increase $H^{*}$; a planted $\mathrm{XOR}_m$ forces $H^{*} \ge m$.
- Flattening / sign-rank (L22): $H^{*}(f) = \Omega(\log \mathrm{sr}_{A|B}(f))$ — the first lower bound independent of $\deg_{\pm}$ (works for $\mathrm{tChow}_{\pm}$ too).
- Counting (L23, L24): nonconstructive existence of $\deg_{\pm}=2$, $H^{*}=\Omega(n)$ functions.
- Explicit flattening separation (L33, L34): $\mathrm{INT}_n$ has $\deg_{\pm}=2$ but $\mathrm{sr}_{x|y}\ge n$, so $H^{*}(\mathrm{INT}_n)\ge\max(3,\tfrac12\log_2 n)$ — the first *named* function provably needing more heads than its threshold degree.
- Localized counting / singleton-column Warren (L35): the **first per-function lower bound beyond flattening** — restrict to a slice where the row acts through $O(H)$ parameters but must shatter many columns, then Warren forces $H=\Omega(\#\text{cols}/\log)$. Gives $H^{*}(\mathrm{INT}_n)=\Omega(n/\log n)$ (positivity-free, so bounds $\mathrm{tChow}_{\pm}$ too), a near-linear explicit separation.
- Shatter-rectangle certificate (L37): the abstract tool. If $g$ admits a shatter-rectangle of order $s$ (coordinate partition $V_{\mathrm{row}}\sqcup V_{\mathrm{col}}$, assignments with $g(\rho_S,\kappa_j)=[j\in S]$), then $\mathrm{tChow}_{\pm}(g)=\Omega(s/\log s)$. Purely combinatorial hypothesis; instances: L35 ($\mathrm{INT}_n$), L36 (disjoint DNF), and the **indexing/multiplexer** $\mathrm{IDX}_k$ (address+memory, output $m_a$) which gives $H^{*}(\mathrm{IDX}_k)=\Omega(2^k/k)=\Omega(N/\log N)$ — and whose $2^k$ terms *share* the address bits, so it lies outside the disjoint-DNF family, showing the partition (not disjointness) is what L37 needs.

## 6. Structure / symmetry

- Closure: output negation and input permutation (L15); full-input complement (L28, via $\alpha\mapsto 1/\alpha$). Single-bit-negation invariance is **open** (holds for $n\le4$ since $H^{*}=\deg_{\pm}$ there; would follow from $\mathrm{tChow}_{\pm}=H^{*}$); the monotone bias only breaks the naive same-atom argument, not necessarily the invariance.
- Junta invariance: $H^{*}$ ignores irrelevant variables (L21).
- **Denominator diversity (L38):** an order-$H$ tangent form with all denominators equal collapses to $D^{H-1}(\theta D + \sum_h N_h)$, an LTF; so $H^{*}(f)\geq 2$ forces at least two *distinct* denominators. Since a denominator is a head's attention pattern, the model's expressive power is denominator (attention-pattern) **diversity**, not head count — heads with a shared pattern add nothing.
- **Gauge invariance of order-2 forms (L40):** order-2 tangent forms carry a $\mathrm{GL}_2$ gauge freedom $(E_1,E_2)=G(D_1,D_2)$, $(M_1,M_2)=G^{-\top}(L_1,L_2)$ that fixes the represented $f$. Rigorous F4 consequence at order 2: the product-sign twist $\mathrm{sign}(D_1D_2)$ is **always removable**, so the only residual positivity obstruction is whether the denominator pencil contains an *admissible* basis (individual positivity $0\notin\mathrm{conv}\{v_x\}$ + one-sided slopes), a gauge-invariant property. Single products $\mathrm{sign}(AB)$ have $H^{*}\le2$ via the difference split $A=E_+-E_-$ (L41). **Order-2-free, the sharp endpoint (focused swing):** rigorous dimension analysis gives $\dim(\text{tChow order-2 variety})=\min(D,4n{-}2)$ ($D=\binom n2+n+1$), full-dimensional iff $n\le6$ (explains the empirical boundary); $V_{\mathrm{adm}}=V_{\mathrm{tChow}}$ are *equidimensional with identical tangent structure*, so **no dimension/transversality argument can decide $n\ge7$**, and $R_{\mathrm{adm}}$ is **not dense** (no perturbation argument either). The residual obstruction is purely **topological covering**: $H^{*}(f)\le2 \iff \Pi(f)\cap\mathcal A\ne\emptyset$ ($\Pi(f)=$ representing 2-planes; $\mathcal A=$ admissibly-spannable planes), immune to dimension and to every order-2 algebraic invariant (σ-twist gauge-removable). Not closable with current ideas; all evidence says it holds. *(A "$\mathrm{tChow}\le r\Rightarrow H^*\le r{+}1$" claim from this swing is INVALID — it builds a degree-2 sum of $r{+}1$ products and mislabels it an order-$(r{+}1)$ tangent form, but the $H^*\le K$ cleared form is degree $K$; they coincide only at $K=2$.)*
- Positivity-free results (F4): $\mathrm{tChow}_{\pm}=H^{*}$ at level $\leq 1$ (L30) and for symmetric $f$ (L31); flattening (L29) and counting (L24) lower bounds need no positivity.
- **Every $H^{*}$ lower bound we have is positivity-free** (threshold degree L6, flattening L22/L29, counting L23/L24, shatter-rectangle L37 — each actually bounds $\mathrm{tChow}_{\pm}$). So no current method can *witness* an F4 gap $\mathrm{tChow}_{\pm}<H^{*}$: the gap, if it exists, is invisible to all known techniques. Strong heuristic evidence (not proof) that positivity is free, i.e. $\mathrm{tChow}_{\pm}=H^{*}$.

## 7. The simple-characterization question ("something like L12")

A clean *exact* closed form for general nonsymmetric $f$ provably is **not** given by alternation counting: the positive-order count $A_{+}$ over-counts (free numerator: $\mathbf 1[x_1\ge x_2]$ has $A_{+}=2 > 1 = H^{*}$), and the signed-order count only bounds $\deg_{\pm}$. Since $H^{*}$ is strictly finer than $\deg_{\pm}$, no linear-order alternation invariant equals it. The honest L12-analogue is $A_{+}$ (exact for symmetric; $H^{*}=A_{+}=\deg_{\pm}$ whenever $\deg_{\pm}=A_{+}$). The exact invariant is the tangential-Chow rank (L16); $H^{*}$ is intrinsically a new measure.

**A second clean family is now pinned (L36):** for a **disjoint monotone DNF** (OR of $s$ pairwise-disjoint ANDs of width $\geq 2$), $H^{*}(f) = \widetilde{\Theta}(s)$ — the head complexity is the *term count*, up to a $\log$ factor. This is an L12-style structural characterization for a genuinely nonsymmetric class: where symmetric $f$ is governed by sign changes (L12), disjoint DNFs are governed by term count. It pairs the monotone-DNF upper bound (L14, $\leq s$) with a matching lower bound (L36, $\geq s/(8\log s)$). The remaining imprecision is only the $\log s$ factor (whether $H^{*}=\Theta(s)$ exactly is open).

## 8. Open problems and the state of the attack

1. **Exact characterization for nonsymmetric $f$** between $\deg_{\pm}$ and $M_{+}-1$: open. Likely no simple closed form (§7).
2. **An explicit (named, non-counting, non-numerical) separation $\deg_{\pm} < H^{*}$:** **SOLVED, and near-linear** (L33-L35, abstracted as L37). The set-intersection function $\mathrm{INT}_n(x,y) = \bigvee_{i=1}^n(x_i\wedge y_i)$ on $2n$ bits has $\deg_{\pm}=2$ (via $\sum_i x_iy_i - \tfrac12$) but cut sign-rank $\mathrm{sr}_{x|y}(\mathrm{INT}_n)\ge n$ (self-contained homogeneous-shattering/VC argument: restrict columns to $y=e_i$, rows $x=\mathbf 1_S$ realize all $2^n$ dichotomies, so the $n$ points are shattered by through-origin halfspaces in $\mathbb{R}^r$, forcing $r\ge n$). Flattening (L22) then gives $n\le (H^{*}+1)2^{H^{*}}+1$, so $n\ge 14$ forces $H^{*}(\mathrm{INT}_n)\ge 3 > 2 = \deg_{\pm}$ — the first rigorous explicit separation, on $\mathrm{INT}_{14}$ ($28$ bits). It is **positivity-free** (L29: same bound holds for $\mathrm{tChow}_{\pm}$, so $\mathrm{tChow}_{\pm}(\mathrm{INT}_{14})\ge 3$ too). This supersedes the old numerical $\neg\mathrm{DISJ}_4$ candidate ($=\mathrm{INT}_4$, same family, $n=8$) — the numerics suggested $H^{*}=3$ at $n=8$ but were unproven; the flattening route proves it for $n\ge 14$. **Strengthening (L34):** since $(H{+}1)2^H{+}1\le 4^H$ for $H\ge 2$, $H^{*}(\mathrm{INT}_n)\ge \tfrac12\log_2 n\to\infty$ while $\deg_{\pm}=2$ — an explicit *unbounded* separation. **Fundamental cap on flattening:** $\deg_{\pm}=O(1)\Rightarrow \mathrm{sr}\le\binom{N}{\le d}=\mathrm{poly}$ (and $\mathrm{sr}_{x|y}(\mathrm{INT}_n)\in\lbrace n,n+1\rbrace$ exactly), so flattening can never certify more than $H^{*}=\Omega(\log N)$ on a constant-degree function. **Near-linear strengthening (L35) — the non-flattening obstruction:** restricting to the singleton-column slice ($x=\mathbf 1_S$, $y=e_j$), the row $\mathbf 1_S$ enters the order-$H$ tangent form through only $2H{+}1$ parameters $(\theta,\alpha_{h,S},\beta_{h,S})$ via a degree-$(H{+}1)$ map, yet must realize all $2^n$ singleton-column dichotomies; Warren then forces $2^n\le(C(H{+}1)n/(2H{+}1))^{2H+1}$, i.e. $H^{*}(\mathrm{INT}_n)\ge\mathrm{tChow}_{\pm}(\mathrm{INT}_n)\ge n/(8\log_2 n)$. This is **positivity-free** and gives the first explicit *polynomial* (near-linear) separation, nearly matching the nonconstructive $\Omega(n)$ of L23/L24. With the DNF upper bound $H^{*}\le n$ (L14), $H^{*}(\mathrm{INT}_n)=\widetilde\Theta(n)$ — pinned up to a $\log$ factor. (Found via a Codex direction-brainstorm.) **The $\log$ gap is hard and now diagnosed** (`lit_survey_round3.md` P1): the $\log$ in the Warren/Goldberg-Jerrum count is *provably necessary* for the abstract sign-family it counts (Bartlett-Maiorov-Meir 1998 bit-extraction VC lower bound), so $\Omega(n/\log n)$ is the **ceiling of the shatter-rectangle method**; reaching $\Omega(n)$ needs a new argument using the tangent-form rigidity (shared $\theta$, tied $N_h/D_h$, the same $P$ on all $2^n$ columns). **Empirically the truth looks linear:** a per-$H$ sweep gives $H^{*}(\mathrm{INT}_n)=2,2,3,4,(\geq 5)$ for $n=2,\dots,6$, i.e. $H^{*}(\mathrm{INT}_n)=n-1$ for $n\geq 3$. **The upper side is now rigorous (L45):** $H^{*}(\mathrm{INT}_n)\le n-1$ via disjoint-term composition (induct the "$+1$ head per disjoint monotone AND" lemma from the $\mathrm{INT}_3$ base, L39) — so $n/(8\log_2 n)\le H^{*}(\mathrm{INT}_n)\le n-1$, the DNF bound $n$ is beaten by one, and the *only* remaining slack is the $\log$ in the L35 lower bound. The $n-1$ saving is provably irreducibly multiplicative: $A_{+}(\mathrm{INT}_n)=2^n-1$ kills the additive weighted-score route.
3. **Does positivity strictly cost (F4: $\mathrm{tChow}_{\pm} < H^{*}$)?** Partial answer: **positivity is provably free** at level $\leq 1$ (L30) and for **all symmetric $f$** (L31, sandwich collapse), and free wherever $\deg_{\pm}(f) = H^{*}(f)$. Computationally **no witness of a gap was found** anywhere ($n\le 4$ exhaustively; $\neg\mathrm{DISJ}_s$ up to $s=5$, including the separation candidate where $\mathrm{tChow}_{\pm}=H^{*}=3$): $\mathrm{tChow}_{\pm}=H^{*}$ in every computed case. **Genuinely open, and a strict gap is plausible** (recalibrated, `lit_survey_round3.md` P2). The small-$n$ numerics ($\mathrm{tChow}_{\pm}=H^{*}$ everywhere computed) are weak evidence, partly because **every $H^{*}$ lower bound the project has (threshold degree, flattening, counting, shatter-rectangle L37) is positivity-free** (§6): no current technique can even *witness* $\mathrm{tChow}_{\pm}<H^{*}$, so "no gap found" is nearly vacuous. Meanwhile the surrounding rank theory shows positivity/one-sidedness constraints **routinely cost, sometimes unboundedly**: nonnegative rank vs rank (Cohen-Rothblum; Shitov, unbounded), symmetric tensor rank (Shitov's disproof of Comon's conjecture), PSD rank (Fawzi et al.), and especially **border rank** — dropping exactness *lowers* rank via a limit, the exact mirror of an F4 cost. So the earlier "positivity is probably free" lean is **not well-supported**; F4 is open with a strict gap a real possibility. Proving one would need a positivity-aware obstruction (a border/nonnegative-rank analogue for the tangential-Chow setting); the level-1 free-numerator trick does not extend through the coupled product $\prod_h D_h$.

**F4 reduces to a product-positivity sign-twist (5-approach workflow).** A multi-angle attack (boundedness, monotone-bias, border-rank, Positivstellensatz, free-via-rounding) found **all five mechanisms dead**, each for a precise reason — gauge invariance ($N_h\to\lambda N_h$ rescales bound and margin together, so no $B/\gamma$ handle); the free mixed-sign numerator absorbs the one-signed apex already at $H=1$ (one atom = every LTF), killing all monotonicity/alternation counts; finite-cube sign-representation is an **open** condition, so Zariski/limit closure adds no Boolean function at fixed order (border rank is the wrong object, gives only upper bounds); an affine $D>0$ at the $2^N$ vertices **is** its own degree-1 positivity certificate (no Putinar/SOS degree blow-up). All five converge on the **single surviving residue**: where no $D_h$ vanishes, $f(x)=1 \iff \sigma(x)R(x)>0$ with $\sigma(x)=\mathrm{sign}\prod_h D_h(x)$; **admissibility forces $\sigma\equiv +1$, while an arbitrary $\mathrm{tChow}_{\pm}$ form lets $\sigma$ be a nontrivial product-of-$H$-halfspaces sign-twist.** And tangent forms are **not additively closed** (appending a head *multiplies* by $D_g$), so the leftover of splitting a mixed-slope factor cannot be absorbed by spending extra heads — structurally the nonnegative-rank$>$rank / border-rank mechanism. So an F4 gap, if real, is exactly a function whose order-$H$ realization *requires* a sign-twisted base product; proving it needs a **$\sigma$-twist dimension invariant** (a hyperplane-cover / nonnegative-rank-style lower bound on admissible heads sensitive to whether $\mathrm{sign}\prod_h D_h$ must be non-constant).

**Targeted test: the $\sigma$-twist is free at order 2.** Implementing the workflow's recommended experiment — *construct* $f$ as the sign of a $\sigma$-twisted order-2 $\mathrm{tChow}$ form (so $\mathrm{tChow}_{\pm}(f)\leq 2$ by construction, with $\mathrm{sign}(D_1D_2)$ genuinely non-constant), then test for an admissible 2-atom — gives, at $n=5$ and $n=6$, **every** such $f$ ($40$ twisted + $25$ control each) admits an admissible 2-atom. This is the *reliable* direction (a representation is found and sign-checked, not a slack-$\approx 0$ infeasibility), so it is solid evidence: at order 2 the $\sigma$-twist costs **nothing** — admissible ($\sigma\equiv+1$) order-2 forms reach every function a sign-twisted order-2 form reaches. Hence **F4 is free at orders $\leq 2$**, and any gap must first appear at $H\geq 3$. The same experiment **at order 3** (construct $f$ from a $\sigma$-twisted order-3 form, test admissible 3-atom) gives the identical verdict: at $n=5,6$, **every** twisted $f$ (30 each) and every control admits an admissible 3-atom — the $\sigma$-twist is **free at order 3 too**. Net F4 status: **free at $H\leq 1$ (L30), symmetric (L31), $\deg_{\pm}=H^{*}$ (L32), and the $\sigma$-twist at $H\leq 3$; open only at $H\geq 4$.** **The lean now reverses toward FREE:** the one surviving gap mechanism, tested by reliable (feasibility-based) construction exactly where it should first bite, costs *nothing* at orders 2 and 3 — so the nonnegative-rank/border-rank analogy (which made a gap *possible in principle*) does **not** transfer to the tangent-form setting at the testable orders. The sharp INT-family test agrees: $\mathrm{tChow}_{\pm}=H^{*}$ for $\mathrm{INT}_3$ and for $\mathrm{INT}_4$ — the latter in the *separation regime* $\deg_{\pm}=2<3=H^{*}$, a non-vacuous F4-free point. So the best current reading is **positivity is most likely free** ($H^{*}=\mathrm{tChow}_{\pm}$, identifying $H^{*}$ with the standard tangential-Chow rank), though a rigorous proof still needs the perturbation-from-arbitrary-to-admissible argument that fails through the coupled product, and a gap at $H\geq 4$ is not excluded.

*(Stress test, and a methodological boundary.* A hunt at $n=7$ with **balanced** $\sigma$-twists ($D_1D_2<0$ on $\approx$ half the cube — the hardest case) flagged apparent gap candidates, but these are **search artifacts**: the verifier failed to find even an *arbitrary* 2-atom for an $f$ whose arbitrary 2-atom exists *by construction*, proving the random-restart search is unreliable at $\geq 128$ points. The decisive control — balanced $\sigma$-twists at $n=5,6$ where the search **is** reliable — gives $0/30$ gaps at each $n$: every balanced-twist $f$ admits an admissible 2-atom. So even the hardest order-2 twists are free, and the numerical search is to be trusted only at $n\leq 6$. This is the correctly-controlled version of the trap that earlier produced the spurious $\neg\mathrm{DISJ}_4$ "gap.")*

## 9. Where the interesting behavior lives

$\deg_{\pm}$, planted parity, flattening, counting, shatter-rectangle are the lower-bound handles. **Separation onset $\approx n=6$:** every degree-2 function on $n\le 4$ has $H^{*}=\deg_{\pm}=2$; the structured $\mathrm{INT}_4$ separates at $n=8$ bits. The whole gap between $H^{*}$ and a standard polynomial invariant is the **tangent (Chow) structure**; whether the *positivity* part adds further power (F4) is open (§8.3).

## 10. Barriers — what current tools provably cannot do (guides the final proof)

The session sharpened the open problems into concrete barriers; breaking any one needs a genuinely new idea, not a refinement.

1. **The flattening/Warren log barrier.** Both per-function lower bounds that beat $\deg_{\pm}$ — flattening (L22) and the shatter-rectangle (L37) — are forms of Warren sign-pattern counting, whose $\log$ factor is *information-theoretically necessary* for the abstract sign-family (Bartlett-Maiorov-Meir bit-extraction, `lit_survey_round3.md` P1). So **no Warren-based method can prove $H^{*}(\mathrm{INT}_n) = \omega(n/\log n)$**; closing the gap to the empirical $\Theta(n)$ needs an argument using the tangent-form *rigidity* (shared $\theta$, tied $N_h/D_h$, one $P$ across all columns), which Warren ignores.
2. **The positivity-blindness barrier (F4).** *Every* $H^{*}$ lower bound we have actually bounds $\mathrm{tChow}_{\pm}$ (§6), so **no current technique can witness $\mathrm{tChow}_{\pm}(f) < H^{*}(f)$**. Proving positivity costs requires a fundamentally positivity-aware obstruction (a border-/nonnegative-rank analogue), of which none is known; proving it is free requires a perturbation from arbitrary to admissible factors that does not increase order, which fails at level $\geq 2$ through the coupled product.
3. **The exact-value barrier.** $H^{*}(\mathrm{INT}_3) = 2$ is provable (L39) *only because* its rigorous lower bound ($\geq 2$, non-LTF) meets its upper bound. For $\mathrm{INT}_n$ with $n \geq 4$ the rigorous lower bound is stuck at $\max(2, \Omega(n/\log n))$ — e.g. flattening gives only $H^{*}(\mathrm{INT}_4)\geq 2$ (since $\mathrm{sr}=4 \leq 13$), so **even $H^{*}(\mathrm{INT}_4) = 3$ is not currently provable**, only numerical. Exact values in the separation regime are gated by barrier 1.
4. **Composition is not subadditive — but ORing a monotone term/DNF *is* cheap (the corner detector).** $H^{*}(f \vee g) \leq H^{*}(f) + H^{*}(g)$ for disjoint $f,g$ is **false** in general: summing the two atom-representations fails because each is bounded-*negative* (not $\approx 0$) off its support, so a single firing input can be cancelled (margin domination needs $m_f m_g > B_f B_g$, impossible as $B \geq m$). The failure is dramatic, not marginal: subadditivity breaks even for two **disjoint LTFs** — Sherstov's intersection of two halfspaces (placed on disjoint blocks) has $\deg_{\pm}=\Omega(n)$, so $H^{*}=\Omega(n)$ while each LTF has $H^{*}=1$ ([arXiv:0910.4224](https://arxiv.org/abs/0910.4224); since $H^{*}\ge\deg_{\pm}$). The precise positive companion (**L45**): if $g$ is a disjoint *monotone DNF* of $s$ terms, then $H^{*}(f\vee g)\le H^{*}(f)+s$ (and dually $f\wedge(\text{disjoint monotone CNF of }s)\le H^{*}(f)+s$). The mechanism is a **monotone corner detector**: the admissible atom $\psi=M/(\gamma+\rho\sum_{i\in A}\alpha^{w_i})$, $\alpha\in(0,1)$, is $\ge0$ everywhere, spikes on the all-ones corner of a term, and is $\approx0$ off it — exactly the "$\approx 0$ when off" normalization that makes L14's OR-of-terms work, now seen as one head per term *added to any base $f$*. **Why this is special to monotone terms:** a one-sided (admissible) denominator is monotone, so a single head can sharply spike only at an *extreme* (all-ones / all-zeros) corner of a coordinate block — never at an interior vertex. So a mixed-polarity term (a parity, or an addressing/equality match) does **not** compose for free; this is the structural reason $\mathrm{IDX}_k$ (L43) and parities are expensive (each address-match is a mixed-polarity corner no single head isolates), while $\mathrm{INT}_n$'s OR-of-monotone-ANDs gives the clean $H^{*}\le n-1$ (L45). (A single term in isolation is always an LTF, $H^{*}=1$; the polarity cost is a *composition* phenomenon.) The flagship: $H^{*}(\mathrm{INT}_n)\le n-1$, with the single "3-ANDs-in-2-heads" saving confined to the $\mathrm{INT}_3$ base (L39) and every later pair a clean $+1$. The $\mathrm{INT}_n$ saving is irreducibly *multiplicative*: $A_{+}(\mathrm{INT}_n)=2^n-1$, so the additive weighted-score route (L25) gives no polynomial bound.
5. **A taxonomy of pair-predicates: linearize (cheap) vs multiplicative ($\widetilde\Theta(n)$).** On the same $2n$ bits $(x,y)$, the head complexity of a pair-predicate is governed by whether it collapses to a *single linear comparison of the two strings*. **Linearizing (constant $H^{*}$):** with super-increasing weights $w_i=2^{i-1}$ the integers $I(x)$ have distinct subset sums, so any function of the difference $d=I(x)-I(y)$ has $H^{*}\le C(G)$ **and bounded sign-rank $\le(C(G)+1)2^{C(G)}+1$** (**L47**, via L25 after flipping $y$ to positive weights, $+$ L22): bitwise equality $\mathrm{EQ}_n=\mathbf 1[d=0]$ has $H^{*}=2$ (L46), greater-than $\mathrm{GT}=\mathbf 1[d>0]$ has $H^{*}=1$, integer/approximate-equality bands $\le2$. L47 is the easy-side delineator (counterpart to the L37 lower-bound certificate): bounded $H^{*}$ here is *certified* by bounded sign-rank, so these predicates provably never separate $H^{*}$ from threshold degree. **Multiplicative ($\widetilde\Theta(n)$):** set intersection $\mathrm{INT}_n=\bigvee_i(x_i\wedge y_i)$ (L35/L45) and subset $x\subseteq y=\neg\mathrm{INT}_n(x,\bar y)$ (L15) — a coordinate-wise conjunction inside an OR that a positive linear order interleaves $2^n$ times. **The separating ingredient is membership/shattering, not mere bilinearity** (corrected after a focused workflow on interior Hamming). $\mathrm{INT}_n$ and $\mathrm{IDX}_k$ are hard because their OR/addressing structure embeds the membership predicate $[j\in S]$ — a shatter-rectangle (L37) — forcing $\widetilde\Theta(n)$. A *lone* bilinear coincidence does not: interior Hamming $\mathrm{HAM}_n{=}1=\mathbf 1[\langle\chi,\xi\rangle=n-2]$ (one step below the equality extreme) has constant threshold degree ($\le4$, via $-(Q-(n-1))^2+\tfrac12$, $Q=$ agreements) and **rigorously $\mathrm{sr}_{x|y}\le n+2$** (explicit $R=h(d)$, $h(j)=\tfrac12-j-\tfrac32(-1)^j$, verified $n\le8$) — only *linear* sign-rank, so flattening yields at most $H^{*}\ge\Omega(\log n)$, never near-linear. Whether even that holds (is $\mathrm{sr}$ unbounded?) is **open with neutral evidence**: all spectral tools are provably $O(1)$ (the $-J$ term dominates; the FSSS bound is $\le2$ under every positive diagonal scaling), and the distance-symmetric Krawtchouk bound "$\ge n$" does *not* transfer to true sign-rank — the EQ control ($\mathrm{sr}(\mathrm{EQ})=3$ exactly, yet invariant rank grows $4,5,6,7$) shows both the invariant bound and the hinge-loss optimizer falsely signal growth. Edge case: $\mathrm{HAM}_2{=}1=\mathrm{PARITY}_4$ exactly (so $H^{*}=4=\deg_\pm$, not a separation). Net: *what attention heads find easy* is comparing two encoded integers (a linear functional of each string); what is provably hard ($\widetilde\Theta(n)$) is the OR/addressing **membership** structure — a single bilinear coincidence is neither.
